# CPBL 今日賽事通知（LINE Bot）

更新日期：2025-11-20  
（目前以冬季聯盟為主，明年例行賽開始前會做更新）

---

## 功能摘要

1. **從中華職棒 CPBL 官網抓取今日賽事資料**  
   自動取得當日所有比賽的場地、對戰組合與開賽時間，並格式化整理。

2. **透過 LINE BOT 發送提醒通知**
   - 每日凌晨 00:00 推播當日賽事資訊
   - 比賽開始前 5 分鐘，發送提醒通知

---

## 使用方法

1. 將自己的 LINE BOT TOKEN 填入環境變數  
2. 使用 Google Cloud Scheduler + Cloud Functions 定時執行程式
